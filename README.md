# Frequency Analysis of Words

Script read text file and return ten popular word

# Quickstart

Example of script launch on Linux, Python 3.5:

```bash

$ python lang_frequency.py <path to file>
Example output:
Word = a, Count=16
Word = and, Count=14
Word = the, Count=10
Word = in, Count=9
Word = questions, Count=9
Word = to, Count=9
Word = of, Count=7
Word = for, Count=6
Word = vote, Count=4
Word = by, Count=4
```
